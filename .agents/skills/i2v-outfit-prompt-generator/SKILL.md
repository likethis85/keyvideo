# I2V 多图穿搭视频提示词生成规约

本技能旨在规范从「模特穿搭主图」与「多张分镜图（幕1~幕5）」转换到 15s 高阶视频生成提示词的分析与编排工作流。通过资产解耦与时空锚定，确保最终生成的视频既能精准保留服装细节，又能展现连贯流畅的镜头动态。

## 核心生成框架 (四大模块)

生成的视频提示词设计应严格遵循以下四大模块解析框架：

### 模块一：视觉资产解耦与锚定 (Asset Decoupling & Anchoring)
在生成过程中，明确分配各参考图的“职能权限”，避免元素揉杂与幻觉：
- **主体锚点 ([@模特穿搭主图])**：赋予最高权重。强制 AI 在整个 15 秒内，严格锁定该图中的人物五官、身材比例、服装款式、色彩及面料细节，确保 100% 一致性。
- **时空锚点 ([@分镜图_幕1] ~ [@分镜图_幕5])**：赋予场控权重。强制 AI 读取每一幕对应的构图比例（特写/中景/全景）、环境背景、光影分布以及模特的初始姿态。

### 模块二：动态微操注入 (Micro-Dynamics Injection)
消除 AI 依图生成的“PPT 幻灯片感”或“黏滞感”，注入符合物理法则的真实动态：
- **摄像机动效 (Camera Motion)**：固定镜头 (Fixed)、缓慢平推 (Slow dolly in)、缓缓拉远 (Slow pull back)、镜头缓慢平滑横移 (Slow panning)。
- **环境动效 (Environment Physics)**：微风轻轻吹动发丝与衣角、侧逆光光影随镜头微小变化、自然光感流转。
- **人物与面料力学 (Character & Fabric Mechanics)**：自然呼吸感、迈步/转身时呈现流线垂坠感 (Fluid drape)、产生真实的物理褶皱。

### 模块三：五幕时间轴激活 (Timeline Activation)
配合传入的 5 张分镜图，按 3 秒一幕的节奏，推演连贯的视觉流：

| 幕次 (时长) | 调用的分镜资产 | 注入的物理与动作激活指令 (Action Prompt) |
| :--- | :--- | :--- |
| **第一幕 (0-3s)** | [@分镜图_幕1] | **[入场与氛围]** 固定镜头开场。以分镜图_幕1为基准，模特从画外从容走入镜头中央（或在画面中保持自然呼吸），微风吹拂发丝，确立出场气场。 |
| **第二幕 (3-6s)** | [@分镜图_幕2] | **[质感与微距]** 镜头极其缓慢地向前平推（Slow dolly in）。基于分镜图_幕2的特写角度，光影微动，清晰凸显面料在真实光照下的高级质感。 |
| **第三幕 (6-9s)** | [@分镜图_幕3] | **[张力与版型]** 基于分镜图_幕3的人物姿态，模特顺势完成下一步连贯动作（如迈出一步或优雅转身），衣摆呈现完美的流线垂坠感（Fluid drape）。 |
| **第四幕 (9-12s)** | [@分镜图_幕4] | **[微动与真实]** 极其缓慢平滑横移镜头（Slow panning）。基于分镜图_幕4姿态，模特进行自然徐缓微动（如微微侧身/视线转动/调整肩膀姿态），衣物随之产生真实的物理垂坠与褶皱。 |
| **第五幕 (12-15s)** | [@分镜图_幕5] | **[定格与种草]** 镜头缓缓拉远（Slow pull back）。模特在分镜图_幕5的最终 Pose 基础上保持高级定格，360度展现完美整体比例，画面淡出。 |

### 模块四：原生音画同构 (Audio-Visual Syncing)
为最终合成的视频自动匹配卡点音效，省去后期的剪辑配音成本：
- **音频 Tag 组合公式**：原生音效：[环境底音] + [动作/材质拟音 Foley] + [情绪 BGM]。
- **输出示例**：原生音效：高级环境白噪音 + 衣服走动时的轻微摩擦声与高跟鞋脚步声 + 契合品牌调性的舒缓纯音乐 BGM。
